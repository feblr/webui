feblr-cli是一个内容生成工具，根据模板（详见feblr-templates）生成资源


# 安装
1. 安装feblr-cli

    ```
    $ npm install -g feblr-cli
    ```

# 使用

1. 查看资源模板

    ```
    $ feblr show

    webpack-ng1-express-mpa
    use webpack angular1 and express

    ```

2. 生成资源

    ```
    $ feblr create demo --template webpack-ng1-express-mpa    
    ```
